Параллельные спуски (h2h)

Пары формируются по общему протоколу квалификации: «сильный с сильным». В дисциплине 2 попытки: во второй попытке участники меняются трассами. При нечётном числе добавляется соперник из организаторов для участника из середины протокола.

Клик по участнику: победа (2 очк.) → второе место (1 очк.) → НФ (0) → сброс

Места в кроссе (этап 2)

До 16 участников в классе — система с двойным поражением; после 16 — модифицированная олимпийская система. Хиты формируются по классам; в одном заезде — участники с высоким и низким рейтингом (змейка по R1).
